# Security Policy

## Supported Versions

This project is a personal learning repository and does not follow a formal versioning scheme yet.  
Security fixes are applied on a best-effort basis to the `main` branch only.

| Version | Supported          |
|---------|--------------------|
| main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security issue or sensitive data exposure in this repository,  
please contact me via email at [ivousd@gmail.com] and include a brief description and steps to reproduce.  
I will review the report as soon as possible and take appropriate action if needed.
